Mini Shell
Languages/Libraries: C
Date Started/Completed: October 2021
Information:
A mini Unix shell built in C. Implements a basic lexer/parser for user commands that breaks them into individual tokens. Also implements a basic signal handler for stopping the shell. Uses child processes using the fork function to allow multiple instances of a shell to run at one time and supports basic shell commands such as ls, cd, echo, cat etc. The program also supports chaining multiple commands using the ";" syntax and piping commands using "|".
