Video Database Platform
Languages/Libraries: Java, HTML, JavaScript, SQL, JDBC, Spring Boot, React.JS
Date Started/Completed: November - December 2021
Information:
A simple representation of a video platform database editor. This application uses mySQL for the backend database storage, connects the database to a Java interface using the JDBC library, and uses Spring Boot and JavaScript for server-side user queries and transactions to the database. The front-end UI is rendered using basic HTML and the React.JS library. More information about this project is in the README of the Git repository, made available upon request.
